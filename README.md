# Java Lab Programs

This repository contains a collection of Java programs developed using Eclipse IDE as part of academic laboratory work and programming practice.

## Overview

The programs included in this repository demonstrate fundamental Java concepts such as:

- Conditional Statements
- Loops and Iterations
- Pattern Printing
- Number Manipulation
- Basic Problem Solving
- Console-Based Input and Output

## Programs Included

1. Leap Year Checker
2. Square Pattern
3. Hollow Square Pattern
4. Even Number Hollow Square Pattern
5. Odd Number Hollow Square Pattern
6. Pyramid Star Pattern

## Technologies Used

- Java
- Eclipse IDE
- JDK (Java Development Kit)

## Learning Outcomes

Through these programs, the following concepts were practiced:

- Java Syntax and Program Structure
- Nested Loops
- Conditional Logic
- Pattern Generation
- Object-Oriented Programming Fundamentals
- Code Organization and Documentation

## How to Run

1. Open the project in Eclipse IDE.
2. Navigate to the desired Java file.
3. Run the file using:

   Run As → Java Application
 
4. View the output in the Eclipse Console.
